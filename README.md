### Deskripsi  
Deskripsi : Aplikasi ini merupakan aplikasi dengan menggunakan bahasa pemrograman kotlin kemudian dilengkapi dengan fitur input nama dan akan menghasilkan output berupa "Halo, nama!" serta dilengkapi dengan mode gelap dan mode terang<br>
### Cara Menjalankan
Cara Menjalankan : 
1. Clone repository ini menggunakan terminal atau CMD https://github.com/muh-ghazii/HelloUI-PemrogramanMobile.git
2. Buka folder repository yang sudah diclone dengan menggunakan IDE Android Studio
3. Tunggu proses Gradle Sync selesai.
4. Lakukan klik Run pada bagian atas atau menggunakan tombol Shift+F10
5. Aplikasi berhasil dijalankan menggunakan emulator bawaan dari IDE
