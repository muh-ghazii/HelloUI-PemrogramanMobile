### Deskripsi  
Deskripsi : Aplikasi ini merupakan aplikasi dengan menggunakan bahasa pemrograman kotlin kemudian dilengkapi dengan fitur input nama dan akan menghasilkan output berupa "Halo, nama!" serta dilengkapi dengan mode gelap dan mode terang<br>
### Cara Menjalankan
Cara Menjalankan : 
1. Clone repository ini menggunakan terminal atau CMD
   ```
   git clone [https://github.com/muh-ghazii/HelloUI-PemrogramanMobile.git]
3. Buka folder repository yang sudah diclone dengan menggunakan IDE Android Studio
4. Tunggu proses Gradle Sync selesai.
5. Lakukan klik Run pada bagian atas atau menggunakan tombol Shift+F10
6. Aplikasi berhasil dijalankan menggunakan emulator bawaan dari IDE <br>
